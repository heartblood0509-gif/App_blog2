API 키 설정

각 서비스에서 발급받은 API 키를 입력하세요

제목, 나레이션, 이미지 생성에 사용됩니다

Google AI Studio에서 발급 |
  1. Google 계정으로 로그인
  2. 위 "Google AI Studio에서 발급" 링크 클릭
  3. "API 키 만들기" 버튼 클릭
  4. 생성된 키를 복사하여 위 입력란에 붙여넣기

무료로 사용 가능합니다 (일일 사용량 제한 있음)

Typecast 고품질 음성 생성에 사용됩니다

Typecast에서 발급 |
  1. https://typecast.ai/developers/api 접속 후 회원 가입
  2. 좌측 목록에서 DEVELOPER TOOLS API 클릭
  3. 상단 API 키 클릭
  4. API키를 복사하여 위 입력란에 붙여넣기

무료 플랜: 월 30,000 크레딧 제공

AI 영상 클립 생성에 사용됩니다 (Hailuo, Kling, Veo 등)

FAL에서 발급 |

API 키 발급 방법

  1. https://fal.ai/dashboard 접속 후 Google 아이디로 가입
  2. 상단 메뉴 Settings 클릭 → API Keys 클릭
  3. Add API Key 버튼 클릭 → 이름 입력 후 발급
  4. 발급된 API 키를 복사하여 메모장 등 안전한 곳에 저장
  5. 저장한 키를 위 입력란에 붙여넣기

API 키는 발급 직후 한 번만 표시됩니다. 이때 복사해서 안전한 곳에 저장해두지 않으면 다시 확인할 수 없어 재발급을 받아야 합니다.

💳 카드 등록 / 결제 방법

  1. https://fal.ai/dashboard/billing 접속 (또는 상단 Settings → Billing)
  2. Add Credits 버튼 클릭
  3. 충전할 금액 선택 후 신용카드 정보 입력 → 결제
  4. 충전된 크레딧은 구매일로부터 365일 내 사용 가능

가입 시 카드 등록 없이 시작 가능하며, 유료 모델 사용 시점에 충전하면 됩니다 (종량제).

빈 칸으로 저장하면 기존 키가 삭제됩니다
