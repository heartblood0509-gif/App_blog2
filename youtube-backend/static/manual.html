주의사항

영상 제작을 시작하기 전에 꼭 확인해야 할 내용입니다.

이 영역에 스크린샷과 설명이 들어갑니다.

API 키 등록하기

영상 제작에는 세 가지 외부 서비스(Gemini · Typecast · FAL)의 API 키가 필요합니다. 아래 항목을 펼쳐서 발급과 입력 방법을 확인하세요.

키가 하나라도 빠지면 해당 단계에서 작업이 멈춥니다. 서버 키로 자동 대체되지 않으니 세 가지 모두 등록해 주세요.
1 제미나이(Gemini) API 키 등록하기
키 발급 방법
  1. Google AI Studio (aistudio.google.com)에 접속해 Google 계정으로 로그인합니다.
  2. 좌측 메뉴에서 "Get API key"를 누른 뒤 "Create API key"를 클릭합니다.
  3. 생성된 키 문자열을 복사합니다.

AI Studio에서 키를 발급받는 스크린샷이 들어갑니다.

사이트에 입력하는 방법
  1. 상단 우측 "API 키 입력" 버튼을 누릅니다.
  2. 설정 페이지의 "Gemini API 키" 칸에 복사한 키를 붙여넣습니다.
  3. 저장 버튼을 눌러 등록을 마칩니다.

설정 페이지의 Gemini 키 입력란 스크린샷이 들어갑니다.

2 타입캐스트(Typecast) API 키 등록하기
키 발급 방법
  1. Typecast (typecast.ai)에 접속해 회원가입 및 로그인을 합니다.
  2. 마이페이지의 "API" 섹션에서 키를 생성합니다.
  3. 생성된 키를 복사합니다.

Typecast 키 발급 화면 스크린샷이 들어갑니다.

사이트에 입력하는 방법
  1. 상단 우측 "API 키 입력" 버튼을 누릅니다.
  2. 설정 페이지의 "Typecast API 키" 칸에 복사한 키를 붙여넣습니다.
  3. 저장 버튼을 눌러 등록을 마칩니다.

설정 페이지의 Typecast 키 입력란 스크린샷이 들어갑니다.

3 FAL API 키 등록하기
키 발급 방법
  1. FAL (fal.ai)에 접속해 회원가입 및 로그인을 합니다.
  2. 대시보드의 "API Keys" 메뉴에서 새 키를 생성합니다.
  3. 생성된 키를 복사합니다.

FAL 키 발급 화면 스크린샷이 들어갑니다.

사이트에 입력하는 방법
  1. 상단 우측 "API 키 입력" 버튼을 누릅니다.
  2. 설정 페이지의 "FAL API 키" 칸에 복사한 키를 붙여넣습니다.
  3. 저장 버튼을 눌러 등록을 마칩니다.

설정 페이지의 FAL 키 입력란 스크린샷이 들어갑니다.

회원가입 & 로그인

Google 계정으로 로그인할 수 있습니다. 첫 가입 시 관리자 승인이 필요합니다.

로그인 화면과 승인 대기 화면이 들어갑니다.

카드 A: AI가 모두 생성

주제만 입력하면 제목·나레이션·이미지·영상까지 AI가 자동으로 만들어줍니다.

주제 입력 → 제목 선택 → 나레이션 → TTS → BGM → 제작 시작 단계별 화면이 들어갑니다.

카드 B: 내가 직접 제공

이미 가지고 있는 대본을 입력하고 줄마다 이미지·영상을 올리거나 AI에 맡길 수 있습니다.

제목·대본 입력 → 줄별 자산 편집 → TTS → BGM 단계별 화면이 들어갑니다.

작업 진행 상태 보기

영상 제작이 시작되면 진행 상태 페이지로 자동 이동합니다.

이미지 생성 → TTS → 영상 조립 단계의 진행 화면이 들어갑니다.

미리보기 & 영상 만들기

생성된 이미지를 확인하고 영상 모드를 선택해 본 제작을 시작합니다.

미리보기 화면과 영상 모드(켄번스/Hailuo 등) 선택 안내가 들어갑니다.

완성본 다운로드

제작이 완료되면 작업이력에서 영상을 다운로드할 수 있습니다.

작업이력 페이지와 다운로드 버튼 안내가 들어갑니다.
