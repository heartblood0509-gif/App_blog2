# Blog Pick 사용자 매뉴얼

이 폴더는 **일반 사용자(쇼핑몰 운영자·마케터 등 비개발자)에게 배포하는 매뉴얼**의 원본입니다.

## 파일 구성

| 파일 | 용도 | 어디에 올리나 |
|---|---|---|
| `INSTALL-WINDOWS.md` | Windows 설치 매뉴얼 | 인스톨러(.exe) 다운로드 페이지 옆 |
| `INSTALL-MAC.md` | Mac 설치 매뉴얼 | 인스톨러(.dmg) 다운로드 페이지 옆 |
| `USER-GUIDE.md` | 사용 매뉴얼 (앱 안 도움말 모달의 원본) | 앱 안에 동일 내용이 들어가므로, 외부 게시는 선택 사항 |

## 배포 전 채워야 할 자리표시자

매뉴얼 안에 `[배포 담당자가 채울 자리]` 로 표시된 곳을 실제 값으로 바꿔주세요.

1. **다운로드 링크** (INSTALL-WINDOWS.md, INSTALL-MAC.md 각각)
   - 인스톨러 파일을 어디서 받는지 안내
   - 가능하면 **파일 크기 + SHA256 해시**도 함께 게시 → 사용자가 파일 무결성 확인 가능

2. **문의 채널** (3개 파일 모두 끝부분)
   - 이메일 / 카카오톡 채널 / 노션 폼 등

## 배포 방법 예시

- **웹사이트 / 노션 / 티스토리 등에 게재**: `.md` 내용을 복사해서 그대로 붙여넣기 (마크다운 지원 플랫폼이면 표·체크박스도 그대로 살아납니다)
- **PDF로 배포**: `pandoc INSTALL-WINDOWS.md -o INSTALL-WINDOWS.pdf` 같은 방식 (선택 사항)
- **GitHub Releases**: 릴리스 노트에 두 매뉴얼 링크 포함

## 사용 매뉴얼(USER-GUIDE.md)과 앱 안 도움말 모달의 관계

- 앱 안의 도움말 모달(`frontend/src/components/HelpModal.tsx`)에는 **이 매뉴얼과 동일한 내용이 JSX로 박혀** 있습니다.
- 매뉴얼 내용을 수정할 때는 **두 곳을 함께 수정**해야 합니다.
  - 원본: `docs/manuals/USER-GUIDE.md`
  - 미러: `frontend/src/components/HelpModal.tsx`
- 단일 원천화(MDX 렌더러 도입 등)는 향후 작업 후보로 남겨둡니다.

## 1차 배포에서 제외된 항목 (추후 보강 대상)

- **스크린샷** — 텍스트만으로 SmartScreen·Gatekeeper·AI Studio 구간을 통과시키기 어려울 수 있음. 사용자 피드백을 보고 2차에 추가.
- **코드서명·공증** — Windows EV 인증서, Apple Developer Program 도입 시 설치 시 보안 경고가 사라지고 Mac 자동 업데이트도 활성화 가능. 매뉴얼 외 별도 결정 사항.
- **다국어** — 한국어만 제공.
