# Blog Pick 사용 매뉴얼

> 앱을 처음 켜셨다면, 이 매뉴얼을 위에서 아래로 차근차근 읽어주세요.
> **5단계만 따라 하시면 글 한 편이 네이버 블로그에 자동으로 올라갑니다.**

---

## ⚠️ 주의사항

이 도구는 **네이버의 자동 로그인·자동 발행** 기능을 사용합니다.

- 네이버 서비스 이용약관과 충돌할 가능성이 있으며, 이로 인한 **계정 제한·차단의 책임은 사용자에게 있습니다**.
- 캡차(보안 문자)·인증 우회 시도는 금지되어 있으므로, **캡차가 뜨면 직접 풀어주세요**.
- 사용 중 네이버에서 이상 접속 경고가 뜨면 **즉시 사용을 중단**하고 휴식 시간을 가지세요.
- **자동 발행은 하루 최대 4편(계정 2개 × 계정당 2편)을 권장합니다.** 글이 더 필요하시다면 블로그픽으로 본문만 생성한 뒤 **직접 네이버 블로그에 복사·붙여넣어** 발행해주세요. 그 이상 자동 발행을 무리하게 돌리면 계정 차단 위험이 커서 권장하지 않습니다.

## ✍️ 어떤 모드를 골라야 하나요?

블로그픽은 글의 성격에 따라 **3가지 모드**를 제공합니다. 본인이 쓰려는 글의 성격을 찾고, 아래 본인 모드 섹션을 따라가세요.

| 모드 | 어떤 글에 좋을까 |
|---|---|
| **후기성 블로그** | 제품을 직접 써본 듯한 자연스러운 후기·체험기·사용 경험 풀어쓰기 |
| **브랜드 블로그** | 우리 브랜드 공식 콘텐츠 — 브랜드 소개, 제품 정보, 가치 입증, 상세 페이지 |
| **AEO 블로그** | 정보성·질문형 글 — 네이버 검색 + ChatGPT·Gemini AI 답변에 양쪽 노출 |

> ⚠️ **1단계에서 모드를 바꾸면 입력값이 초기화됩니다.**
> 제품·서사·말투·템플릿·프로필 같은 모드 전용 입력은 모드 변경 시 사라집니다. 모드를 확정한 뒤 진행하세요.

## 📝 후기성 블로그 5단계

> **사전 준비 · 제품 등록 (필수)**
> "제품 관리" 페이지에서 발행할 제품을 먼저 등록해주세요. 제품이 등록돼 있지 않으면 1단계에서 진행이 막힙니다.

### 1단계 · 글 구조 (제품 + 서사 + 말투)

세 가지를 차례로 선택합니다.

1. **제품 선택** — 발행할 제품을 하나 이상 선택 (다중 가능)
2. **서사 구조 선택** — 2가지 중 하나

| 서사 | 흐름 / 어울리는 글 |
|---|---|
| **감정 선공형** | 스트레스→악화→시도→깨달음→제품 발견→변화 (11단계). 감정이 강한 후기·회복 스토리에 좋습니다. |
| **결론 선공형** | 현재 상태→과거 문제→시도→실패→깨달음→변화 (9단계). 결론부터 보여주고 풀어쓰는 구조. |

3. **말투 선택** — 존댓말 / 반말 / 음슴체 중 하나 (각 말투 예시는 직접 커스터마이징 가능)

### 2단계 · 글 설정 (메인 키워드 + AI 추천)

- **메인 키워드 (필수)** — 검색 노출에 직결됩니다. "겨울철 두피 관리" 같은 한 줄. **이 모드에서는 메인 키워드가 없으면 다음 단계로 진행할 수 없습니다.**
- **무엇에 대해 쓰고 싶나요? (선택)** — 비워두고 두 가지 도구를 활용할 수 있습니다:
  - **✨ AI 추천** — 메인 키워드 + 제품 정보로 AI가 글 주제를 자동 제안
  - **📝 가이드 질문에 답하기** — 3가지 질문(누구의 이야기? / 어떤 고민·증상? / 어떤 회복·성과?)에 답하면 페르소나·주제가 자동으로 정리됩니다

### 3단계 · 제목 선택

제품 컨텍스트 + 선택한 말투·서사를 바탕으로 제목 후보가 자동 생성됩니다. 마음에 드는 걸 고르세요.

### 4단계 · 본문 생성

- 후기 톤의 본문이 자동 생성됩니다. 보통 **30초~2분** 소요. 닫지 말고 기다리세요.
- 에러가 나면: 보통 **API 키 한도 초과** 또는 **인터넷 끊김**. API 키 확인 후 재시도.

본문은 SmartEditor(네이버 블로그 에디터와 같은 UI)에서 자유롭게 편집 가능. **편집 시 절대 누르지 말 것**:

> 🚨 **Cmd+A (Mac) / Ctrl+A (Win) → Delete**
> 전체 선택 후 한 번에 지우면 편집기가 **흰 화면**이 되는 알려진 문제가 있습니다. 통째로 지우려면 모달을 닫고 처음부터 다시 생성하세요.

### 5단계 · 발행

1. 발행할 **네이버 계정 선택**
2. **카테고리·태그·공개 설정** 확인
3. **"발행"** 버튼 클릭
4. 자동화 창이 잠깐 떴다가 닫힙니다 → **그 동안 키보드·마우스 건드리지 마세요**
5. 발행 완료되면 토스트 메시지와 함께 **글 URL** 이 표시됩니다

## 🏷️ 브랜드 블로그 5단계

> **사전 준비 · 브랜드 프로필 등록 (필수)**
> "브랜드 프로필" 페이지에서 우리 브랜드의 톤·스타일·자주 쓰는 표현을 미리 정의해주세요. 없으면 1단계에서 등록 안내 모달이 뜹니다.

### 1단계 · 글 구조 (브랜드 프로필 + 템플릿)

1. **브랜드 프로필 선택** — 등록해둔 프로필 중 하나
2. **글 템플릿 선택** — 4가지 중 하나

| 템플릿 | 설명 |
|---|---|
| **소개글** | 브랜드 소개, 문화, 가치 — "우리는 어떤 곳인가" |
| **정보성글** | 제품 정보·활용법·사용 사례 (구조형 / 사용자 직접형 두 변형 중 선택) |
| **가치 입증글** | 신뢰도·후기 인용·검증 — "왜 우리를 믿어야 하는가" |
| **상세 페이지글** | 상세 사양·비교·기술 (변형 옵션 있음) |

### 2단계 · 글 설정 (주제 + 키워드)

- **주제 (선택)** — 비워두면 AI가 알아서 정합니다
- **메인 키워드 (선택, 권장)** — 검색 노출을 신경 쓰신다면 입력

### 3단계 · 제목 선택

브랜드 톤 + 선택한 템플릿을 바탕으로 제목 후보가 자동 생성됩니다.

### 4단계 · 본문 생성 — 글 적합성 자동 점검

> ⚠️ **생성 직전 AI가 주제·템플릿·브랜드의 궁합을 검사합니다.**
> 맞지 않다고 판단되면 경고 모달이 뜹니다. 3가지 선택:
>
> ① **제안 받아들이고 주제 바꾸기** (가장 안전)
> ② **이전 단계로 돌아가기**
> ③ **무시하고 그대로 생성**

통과되면 브랜드 톤의 본문이 자동 생성됩니다. 보통 **30초~2분** 소요.

> 🚨 **편집 시 절대 누르지 말 것 — Cmd+A / Ctrl+A → Delete**
> 전체 선택 후 한 번에 지우면 편집기가 흰 화면이 됩니다. 통째로 지우려면 모달을 닫고 처음부터 다시 생성하세요.

### 5단계 · 발행

네이버 계정 선택 → 카테고리·태그·공개 설정 → 발행. 자동화 창이 떴다 닫히는 동안 **키보드·마우스 건드리지 마세요**. 완료되면 글 URL이 표시됩니다.

## 🔍 AEO 블로그 5단계

> **사전 준비 · AEO 프로필 등록 (필수)**
> AEO 프로필 페이지에서 도메인 영역과 전문성 기준을 먼저 설정하세요.

### 1단계 · 글 구조 (AEO 프로필 + 글 의도)

1. **AEO 프로필 선택** — 등록해둔 프로필 중 하나
2. **글 의도 선택** — 5가지 중 하나. 어떤 질문에 답하는 글인지 정합니다

| 의도 | 설명 / 예시 |
|---|---|
| **AI에게 맡기기** | 기본값. 의도를 AI가 자동 추정. 어떤 걸 골라야 할지 모르면 이걸 선택하세요. |
| **정보 탐색형** | 개념·기본 정보 중심. "두피 건선이 뭔가요?" 같은 글. |
| **비교 검토형** | 차이·기준·장단점 중심. "두피 샴푸 A vs B" 같은 글. |
| **구매 전 고민형** | 체크 포인트·기준·실수 방지. "두피 케어 제품 사기 전 알아야 할 것" 같은 글. |
| **문제 해결형** | 원인·해결법·관리 중심. "두피 가려움이 안 잡힐 때" 같은 글. |

"AI에게 맡기기" 외 4개 중 하나를 고르면 **글 의도 자동 반영(Intent Mode)**이 켜져서, 본문 어휘·구조·이미지가 의도에 맞춰 자동으로 조정됩니다.

### 2단계 · 글 설정 (주제 + 키워드)

- **주제 (선택)**
- **메인 키워드 (선택, 권장)** — 검색 노출을 신경 쓰신다면 입력

### 3단계 · 제목 선택

선택한 의도 1개의 각도 안에서 **5개 제목이 변주**되어 생성됩니다. 의도가 흩어지지 않게 자동 통제됩니다.

### 4단계 · 본문 생성

"AI에게 맡기기"가 아닌 의도를 고르신 경우 자동 조정 사항:

- 도입·어휘·주의점·FAQ 패턴이 의도별로 강제됨 — 글이 의도에서 벗어나지 않게
- 이미지 12장 → **4장 하드 캡** (정보성 글의 미니멀 정책)

"AI에게 맡기기" 선택 시 기존 동작(제한 없음)으로 생성됩니다. 보통 **30초~2분** 소요.

> 🚨 **편집 시 절대 누르지 말 것 — Cmd+A / Ctrl+A → Delete**
> 전체 선택 후 한 번에 지우면 편집기가 흰 화면이 됩니다. 통째로 지우려면 모달을 닫고 처음부터 다시 생성하세요.

### 5단계 · 발행

네이버 계정 선택 → 카테고리·태그·공개 설정 → 발행. 자동화 창이 떴다 닫히는 동안 **키보드·마우스 건드리지 마세요**. 완료되면 글 URL이 표시됩니다.

## 🚨 자주 막히는 함정 (꼭 한번 읽어주세요)

**네이버 2단계 인증을 켜뒀어요**
→ 발행할 때 OTP 입력 창이 떠요. 휴대폰 보안 앱을 켜두세요.
→ **2단계 인증은 끄지 않는 것을 권장**합니다 (계정 보호 우선).
→ 자동 발행 빈도가 잦으면 발행 전용 부계정을 따로 만드시되, 그 부계정도 2단계 인증은 유지하세요.

**네이버가 "비정상 접속"으로 차단했어요**
→ 같은 IP에서 너무 자주 자동 발행하면 일시적으로 막힙니다.
→ **하루 10편 이하** 권장. 차단되면 한동안 사용을 멈추고 자연스럽게 회복되길 기다리세요.

**SmartEditor가 흰 화면이 됐어요**
→ 위의 ⚠️ 경고(Ctrl+A → Delete)를 안 지키신 경우입니다.
→ 모달을 닫고 글을 다시 생성하세요. 저장된 이미지는 보관함에 남아 있을 수 있습니다.

**발행이 중간에 멈췄어요**
→ 캡차가 떴을 가능성이 큽니다. 자동화 창에 캡차가 보이면 **직접 풀어주시면** 이어집니다.
→ 자동화 창을 닫지 마세요.

**API 키 한도 초과 에러가 나요**
→ Gemini는 무료 한도가 있습니다.
→ 다음 날 자동으로 풀리거나, Google Cloud에서 결제 연결로 한도를 늘릴 수 있습니다. 자세한 요금: https://ai.google.dev/gemini-api/docs/billing

## 💾 내 데이터 백업·다른 PC로 옮기기

블로그픽의 모든 데이터(등록한 제품, 브랜드 프로필, 보관함 이미지 등)는 다음 폴더에 있습니다.

| OS | 데이터 위치 |
|---|---|
| **Windows** | `%APPDATA%\app-blog2-desktop\` (파일 탐색기 주소창에 그대로 입력) |
| **Mac** | `~/Library/Application Support/app-blog2-desktop/` (Finder → 이동 → 폴더로 이동) |

### 백업하기

1. 위 폴더를 통째로 **압축** (zip)
2. 클라우드·외장하드 등 안전한 곳에 보관

### 다른 PC로 옮기기 (PC 교체 시)

1. 기존 PC에서 위 폴더를 압축 → 새 PC로 옮기기
2. 새 PC에 Blog Pick **설치 → 종료** (한 번 실행해서 폴더 생성 후 종료)
3. 새 PC의 동일 경로 폴더에 백업본을 풀어 덮어쓰기
4. 새 PC에서 Blog Pick 실행

> ⚠️ **주의 사항**
