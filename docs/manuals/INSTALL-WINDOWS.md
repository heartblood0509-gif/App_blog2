# Blog Pick 설치 매뉴얼 — Windows

> 이 문서는 **Windows 컴퓨터**에서 Blog Pick을 처음 설치하시는 분을 위한 안내입니다.
> 위에서부터 한 줄씩 따라하시면 30분 안에 첫 글 발행까지 가능합니다.

---

## ⚠️ 주의사항

이 도구는 **네이버의 자동 로그인·자동 발행** 기능을 사용합니다.

- 네이버 서비스 이용약관과 충돌할 가능성이 있으며, 이로 인한 **계정 제한·차단의 책임은 사용자에게 있습니다**.
- 캡차(보안 문자)·인증 우회 시도는 금지되어 있으므로, **캡차가 뜨면 직접 풀어주세요**.
- 사용 중 네이버에서 이상 접속 경고가 뜨면 **즉시 사용을 중단**하고 휴식 시간을 가지세요.
- **자동 발행은 하루 최대 4편을 권장합니다** — 계정 2개 × 계정당 2편이 안전선입니다. 글이 더 필요하시다면 블로그픽으로 본문만 생성한 뒤 **직접 네이버 블로그에 복사·붙여넣어** 발행해주세요. 그 이상 자동 발행을 무리하게 돌리면 계정 차단 위험이 커서 권장하지 않습니다.

이 안내에 동의하신 분만 다음 단계로 진행해주세요.

---

## 🧾 준비물 체크리스트

설치 시작 전 아래가 모두 준비됐는지 확인하세요.

- [ ] **Windows 10 이상** (Windows 11 권장)
- [ ] **네이버 계정** (글을 올릴 네이버 블로그 계정)
- [ ] **Gemini API 키** (무료 발급, 아래 1단계에서 안내)
- [ ] **인터넷 연결** (안정적인 연결 필요)
- [ ] (선택) **공식 배포처에서 받은 설치 파일** (.exe)

---

## 1단계. Gemini API 키 발급 (5분)

블로그픽이 AI로 글을 쓰려면 Google의 Gemini API 키가 필요합니다.

1. 웹브라우저에서 **`https://aistudio.google.com/app/apikey`** 접속
2. **Google 계정으로 로그인** (기존 Gmail 계정 사용 가능)
3. **"Create API key" (API 키 만들기)** 버튼 클릭
4. (창이 비어 있고 키 생성 버튼이 안 보이는 경우) **"Projects" → "Import projects"** 로 들어가 기존 프로젝트를 가져온 뒤 다시 시도
5. 발급된 키(`AIzaSy...`로 시작하는 긴 문자열)를 **메모장이나 안전한 곳에 임시 저장**
   - ⚠️ 키는 비밀번호처럼 다뤄야 합니다. 절대 SNS·공개 채팅에 올리지 마세요.

**💰 비용 안내**
- 무료 한도 내에서 사용 가능합니다.
- 사용량이 많아지면 결제 설정이 필요할 수 있습니다.
- 현재 요금 정책: https://ai.google.dev/gemini-api/docs/billing

---

## 2단계. 설치 파일 다운로드

공식 배포처에서 **`Blog-Pick-Setup-XXX.exe`** 파일을 받으세요.

> 📥 **다운로드 링크:** `[배포 담당자가 채울 자리]`

**Tip**
- 32bit / 64bit 구분이 안 보이면 **64bit를 받으면 됩니다** (요즘 PC는 거의 다 64bit).
- 다운로드한 파일의 **크기·해시값**이 배포처에 적힌 값과 같은지 한번 확인하면 안전합니다.

---

## 3단계. 설치 실행 ⚠️ 여기서 가장 많이 막힙니다

다운로드한 `.exe` 파일을 **더블클릭**하면 아래와 같은 **파란 창**이 뜰 수 있습니다.

> 🔵 **"Windows에서 PC를 보호했습니다"**
> 인식할 수 없는 앱이 시작되지 않도록 했습니다. (Microsoft Defender SmartScreen)

**👉 공식 배포처에서 받은 파일이 맞다면 아래 절차로 여세요:**

1. 창에 있는 작은 글씨 **"추가 정보"** 클릭
2. 아래쪽에 새로 나타나는 **"실행"** 버튼 클릭

> 🛡 이 경고는 새로운 앱에 대해 Windows가 항상 띄우는 일반적인 안내입니다.
> 단, **출처가 불분명한 파일이라면 절대 실행하지 마세요**.

---

## 4단계. 설치 진행

설치 마법사가 시작됩니다.

- 설치 위치: **기본값 그대로 두기**
- "바탕화면 바로가기 만들기": **체크 권장**
- 설치가 끝나면 자동으로 Blog Pick이 실행됩니다

---

## 5단계. 첫 실행 시 권한 허용

처음 실행하면 다음 두 종류의 알림이 뜰 수 있습니다.

### 5-1. Windows 방화벽 알림

> "이 앱이 네트워크 통신을 허용할까요?"

- **"개인 네트워크"** 만 체크하고 **"액세스 허용"** 클릭
- 공용 Wi-Fi(카페·공항)에서는 굳이 허용하지 않아도 됩니다.
- AI(Gemini)와 통신하려면 이 권한이 꼭 필요합니다.

### 5-2. 백신 프로그램이 차단할 경우

V3, 알약, Windows Defender 등이 의심한다며 차단하는 경우가 있습니다.

- 공식 배포처에서 받은 파일이 확실하다면, 백신의 **"예외 등록"** 또는 **"신뢰할 수 있는 항목"** 으로 추가
- 잘 모르겠다면 배포처에 먼저 문의

---

## 6단계. 로그인 + API 키 등록

Blog Pick이 열리면 로그인 화면이 보입니다.

1. 가입 시 사용한 **이메일/비밀번호**로 로그인
2. 로그인 후 화면 우측 상단의 **🔑 열쇠 아이콘** 클릭 → "API 키 설정" 페이지
3. 1단계에서 저장해둔 **Gemini API 키**를 입력칸에 붙여넣기
4. **"저장"** 클릭

---

## 7단계. 네이버 블로그 계정 등록

발행할 네이버 계정을 등록합니다.

1. 네이버 ID/비밀번호 입력
2. 비밀번호는 컴퓨터에 **암호화되어 저장**됩니다 (외부로 전송되지 않음)
3. ⚠️ **네이버 2단계 인증**을 켜둔 계정이면 발행 시 **OTP 입력 창**이 뜰 수 있습니다 → 휴대폰을 곁에 두세요
4. 보안을 위해 **2단계 인증은 계속 켜둘 것을 권장**합니다 (해제하지 마세요)

---

## 8단계. 첫 글 발행해보기

이제 준비 완료. 앱 상단의 **❓ 도움말 버튼**을 클릭하면 사용 매뉴얼이 나옵니다.
거기 안내대로 한 편 발행해보세요.

---

## 🔄 자동 업데이트 안내

- **Windows는 자동 업데이트가 활성화**되어 있습니다.
- 새 버전이 나오면 앱이 자동으로 알려주고, 클릭 한 번으로 업데이트됩니다.
- 따로 새 설치 파일을 다시 받지 않아도 됩니다.

## ❓ 자주 묻는 질문 (FAQ)

**Q. 설치 파일이 안 열려요**
A. "추가 정보 → 실행" 절차를 다시 확인하세요. 그래도 안 되면 다운로드 출처(공식 배포처)가 맞는지 다시 확인.

**Q. 백신이 바이러스라고 경고해요**
A. 공식 배포처에서 받은 파일이 맞다면 백신의 예외 등록으로 처리할 수 있습니다. 출처가 불확실하면 절대 실행하지 말고 먼저 배포처에 문의하세요.

**Q. 다시 설치하고 싶어요**
A. 제어판 → "프로그램 추가/제거" → "Blog Pick" 삭제 후 새 설치 파일로 다시 설치하면 됩니다. **이때 등록한 계정·이미지·보관함은 그대로 유지됩니다** (앱만 다시 깔리는 것이라).

**Q. Gemini API 무료 한도가 초과됐어요**
A. 다음 날 자동으로 풀리거나, Google Cloud에서 결제 연결로 한도를 늘릴 수 있습니다. https://ai.google.dev/gemini-api/docs/billing 참고.

**Q. 발행 중 캡차(보안 문자)가 떴어요**
A. 자동화 창에 캡차가 보이면 **직접 풀어주세요**. 자동 우회는 약관 위반입니다.

**Q. 회사 컴퓨터에서 쓰려고 합니다. 방화벽이 막아요**
A. 회사 IT팀에게 다음 도메인을 허용해달라고 요청하세요:
- `*.googleapis.com` (Gemini API)
- `*.naver.com`, `*.pstatic.net` (네이버 발행)
- `github.com`, `objects.githubusercontent.com` (자동 업데이트)

## 🗑 완전 삭제 (데이터까지 모두 지우고 싶을 때)

> ⚠️ **경고**: 아래 절차를 진행하면 **등록한 네이버 계정, API 키, 생성한 이미지, 보관함**이 전부 사라집니다.
> 백업이 필요하다면 사용 매뉴얼의 "백업·PC 이전" 섹션을 먼저 보세요.

1. 제어판 → **"프로그램 추가/제거"** → "Blog Pick" 선택 → "제거"
2. 파일 탐색기를 열고 주소창에 **`%APPDATA%\app-blog2-desktop`** 입력 → 엔터
3. 열린 폴더를 **통째로 삭제**
4. (필요 시) 같은 방법으로 **`%LOCALAPPDATA%\app-blog2-desktop`** 도 확인 후 삭제
